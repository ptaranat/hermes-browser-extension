PUBLIC SETUP

Connect to Hermes

  1. Install Hermes Agent/Desktop and keep the local gateway running.
  2. Clone this repo, build it, and load the dist/ folder unpacked.
  3. Open the Hermes extension side panel from your browser toolbar.
  4. Click Connect to Hermes, or use Manual setup with your local API key.
  5. Approve locally if prompted, then open a normal web page.
  6. Ask Hermes about the active page context.

Public alpha: load unpacked, local-first, read-only browser context.
